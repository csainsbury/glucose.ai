Project Overview

Development of transformer-based models for medical event prediction using MEDS format data from Stanford Healthcare. The project evolved from single discriminative model (ASCEND) to include a generative model (ASCENDGPT) with shared architecture.

Entries 10-15 June 23-24, 2025

ASCEND Model Implementation

Life2Vec Methodology Applied

Implemented transformer-based foundation model for cardiovascular outcome prediction with medical-specific adaptations.

Technical Architecture

Model Configuration:
- 6 transformer layers
- 512 embedding dimension
- 8 attention heads
- 27M parameters
- Max sequence length: 2048

Key Components

  • Feature Selection: Top 5,000 from 55,672 available
  • Tokenization: Quintile binning for 475 continuous features
  • Time Encoding: Multi-scale temporal features
  • Pre-training: MLM + SOP tasks

Multi-Scale Time Encoding

Innovative approach combining absolute time, relative time, patient age, and temporal patterns (hour/weekday/season)

Entries 16-19 June 24, 2025

Model Fine-tuning & Critical Discovery

Initial Results

Composite outcome prediction achieved 0.953 AUC-ROC but with 95.6% positive prevalence - indicating a flawed temporal framework.

Critical Discovery

Temporal Framework Issue

Model was predicting historical presence, not future occurrence. Fixed by using first clinical event (not birth) as baseline.

Entries 20-30 June 24, 2025

Multi-Label Optimization & Performance Breakthrough

Multi-Label Framework

Implemented simultaneous prediction of Death, MI, Stroke, and Heart Failure with proper temporal windowing.

Key Optimizations

  • Random Prediction Points: More realistic than fixed points
  • Extended Baseline: Up to 5 years of patient history
  • Class Weight Balancing: Critical for 0.3-3.3% event rates
  • Memory Optimization: Gradient accumulation, dynamic padding
  • Learning Rate Schedule: Warmup + cosine annealing

Performance Results

76% Relative Improvement

Macro AUC improved from 0.463 to 0.816 through optimization

Entry 35 January 31, 2025

ASCENDGPT - Generative Medical Sequence Model

Architectural Innovation

Created modular monorepo structure with 60-70% shared codebase between discriminative and generative models.

Multi-Head Generation

Simultaneous prediction of next event, timing, value quantile, and event type

Key Components

  • Flexible Embeddings: Configurable for causal/bidirectional
  • Advanced Attention: Flash, rotary, ALiBi support
  • Generation Heads: Event, time, value, type prediction
  • Training: Multi-task learning with weighted losses
